Polygon Area Calculator
=======================

Simple script to calculate the area of common polygons (triangle, square, rectangle, circle, regular polygon).

Usage
-----

- Run the script with Python 3:

	python poligon-area-calculator.py

- Follow the interactive prompts to choose a shape and provide dimensions.

Requirements
------------

- Python 3.x (no external libraries required)

Example
-------

- Choose `triangle`, then enter base and height to get the area.

Notes
-----

- The script is intended as a small learning project. Inputs are read from the console.
